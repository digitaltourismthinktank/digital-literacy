Foundations · the learning surface

A kit of interactive parts every module draws from.

The course leans on doing over watching. Rather than design seventy modules by hand, we build a small set of reusable interactions, then each module is assembled from them. This keeps the experience contained and consistent, and pushes engagement well past a video and a quiz.

Contained

Two-way
Predict, choose or adjust before the answer appears. The interaction earns the reveal.
Consistent
Same parts, same rhythm, same slot. Learn the shape once, use it across every track.

Definition
CSV: comma-separated values. A plain text data file, one row per line, values split by commas.
Key idea
AI reads the text underneath your spreadsheet, not the grid you see on screen.
Note
Spreadsheet apps can hide problems by auto-formatting dates as you open the file.
Caution
A single missing value can silently shift every column after it.

Group 02
Reveal & disclose
Define on demand Inline
Use for defining a term the first time it appears without breaking the reading flow.

Most data faults are not bugs in the model. They are problems in the , the small character that separates one value from the next. Get it wrong and the meaning shifts.

Delimiter
The character that marks where one value ends and the next begins. In a CSV it is the comma. The trouble starts when a value contains the delimiter itself.
Go deeper Accordion
Use for optional depth, so a confident learner moves on while a curious one expands.
The parser counts commas to find columns. A name like Naoussa, Paros adds a comma the parser was not told to ignore, so the row gains a column and every value after it slides one place to the right.
Wrap the value in quotation marks. "Naoussa, Paros" is read as one value, comma and all. Most exports do this automatically, but not all, which is exactly why it is worth checking.
Try, then reveal Two-way
Use for making the learner commit to a prediction before the explanation lands.
A row reads 2026-07-03,Seaside Cabin,280.00 in a file whose columns are date, property, guests, revenue. What will the model think the guest count was?
280 guests. The guests value is missing, so 280.00 slid into its place and revenue was left empty. One absent value corrupted two columns at once, with no error.

Group 05
The paged module

A module is no longer one scrolling page. It is a paged lesson: one part per frame, advanced with back and next inside a fixed-height stage, with nothing scrolling within a frame. Three parts are specific to that model and assemble every module: the designed cover, the stage and its controls, and the Your turn block that closes the lesson.

Lesson cover Frame 1
Use for the opening frame of every module. The square orientation video is a deliberate centrepiece, not an edge thumbnail, and stays optional and dismissible.
